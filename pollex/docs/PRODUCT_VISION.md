# Product Vision

Pollex is adaptive UI infrastructure powered by Tactus, an engine that learns from user behavior and safely adapts interfaces inside a product's design system.

Phase 1 focuses on the backend engine loop:

1. Observe lightweight interaction events.
2. Understand behavior through deterministic UX profiles.
3. Propose small, safe adaptations.
4. Guard every proposal with policy and design-system checks.
5. Resolve an adaptation decision for SDK callers.
6. Measure outcomes for future learning.
7. Remember profile state without creating per-user UI rules.

Pollex is not an analytics dashboard, feature flag platform, manual experiment system, heatmap product, or replay tool.
